# Control Systems

Interactive control toolbox for learning control theory and designing system.

![screenshot](https://i.imgur.com/PdS5xZr.png)

## Features:

- simply build system from expression
- calculate parameters like Ts, overshoot
- real-time diagram
- plot rlocus
